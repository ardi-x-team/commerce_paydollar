Module: Commerce PayDollar DataFeed Controller
Author: Quintus Leung

Description
===========
* Provide DataFeed feature for Commerce PayDollar.



Requirements
============
* Commerce PayDollar


Setup
=====
* There's nothing to be setup in the Drupal. 
  The only thing need to setup is from the backend of the PayDollar it
  self by inserting this url below for DataFeed absolute to your domain.
  ie: http://<your-site.com>/paydollar/datafeed
