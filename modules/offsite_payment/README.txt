Module: Commerce PayDollar Offsite
Author: Pratomo Ardianto and Quintus Leung

Description
===========
* Provide Offsite payment for Commerce PayDollar.



Requirements
============
* Commerce PayDollar


Setup
=====
* Go to Store -> Configuration -> Payment methods
   -> enable PayDollar Offsite Payment.

* Edit PayDollar settings and select whether you want to operate in the test
  or the live environment, input your merchant id and hash password (provided
  by PayDollar).
